## 介绍

Jindo 作为阿里云 OSS 数据湖计算存储场景中的湖加速方案，完全兼容 Hadoop 生态，并针对 Hadoop 大数据中 Spark、Hive、Flink、Presto 等组件进行了深度优化。同时，开发者也可以在自建生产环境部署 JindoSDK 快速集成 OSS 和 JindoFS 等存储，获得更好的性能。
